## Diwali Sales Data Analysis.

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Diwali sales data to understand customer behavior and purchasing patterns.

## Tools Used
1) Python
2) NumPy
3) Pandas
4) Matplotlib
5) Seaborn

## Key Insights
Female customers contribute higher sales compared to males.
The 26–35 age group is the highest spending segment.
Female dominance in sales is consistent across all states.
Uttar Pradesh records the highest total sales.
The most purchased product categories are Food & Clothes.

## Conclusion
The analysis highlights females aged 26–35 as the most valuable customer segment during Diwali sales, suggesting targeted marketing opportunities.
